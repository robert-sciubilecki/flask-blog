About

Welcome to my coding journey, a fusion of news and projects. I'm proficient in Python, JavaScript, HTML, CSS, and React. A self-taught developer, I've invested time and effort to grasp coding fundamentals.

My previous life as a musician was prosperous and enjoyable, but I couldn't resist the allure of coding. I'm now eager to collaborate with experienced developers, just as I did with fellow musicians.

This blog serves as a portfolio piece. It's a full-stack application with a Flask backend, featuring user registration and comments functionality. I've taken steps to ensure security with password hashing and user authorization via Flask-Login. All data, including posts, comments, and user information, is stored in an SQLite database. It also has RESTful API for CRUD operations. The simple frontend is my own creation, without the use of Bootstrap or pre-made templates. While JavaScript plays a minor role in this project, I'm always open to exploring its potential.

Join me on my coding journey, collaborate on exciting projects, and connect with like-minded individuals. If you have project ideas or are looking for a passionate coder, don't hesitate to reach out. Let's build something amazing together!
